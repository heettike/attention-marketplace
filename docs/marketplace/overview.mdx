## how suppliers are vetted

1. **application** - suppliers apply with portfolio and results
2. **review** - noice team reviews track record
3. **trial** - new suppliers start with limited visibility
4. **featured** - top performers get featured status

## pricing

suppliers set their own rates. typical ranges:

| category | price range |
|----------|-------------|
| kols | $3k - $50k per post |
| market makers | $50k - $200k setup |
| trenching | $3k - $15k per campaign |
| video | $5k - $20k per video |
| podcasts | $2k - $10k per episode |
| newsletters | $5k - $25k per placement |

## results tracking

every supplier shows:
- **views per $** - reach efficiency
- **past work** - links to actual posts/content
- **client results** - before/after metrics
